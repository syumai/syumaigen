# syumaigen

* A CLI tool to generate syumai's avatar image.
* The avatar used in this command was designed by @tanakaworld.

## Features

* [WIP] Randomize color generation

## Installation

```console
go get -u github.com/syumai/syumaigen/cmd
```

## Usage

```sh
syumaigen > syumai.png
```
